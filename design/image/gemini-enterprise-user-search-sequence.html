sequenceDiagram autonumber participant Browser as ブラウザ participant Gemini as Gemini Enterprise participant STS as Google STS(sts.googleapis.com) participant AgentSearch as Gemini Enterprise Agent Search participant PolicyGuard as Cloud Run Policy Guard participant DataStore as Search Data Store (Discovery Engine) Note over AgentSearch,DataStore: 検索時は格納済みデータと ACL メタデータを使って可否判定し、元システムへはリアルタイム問い合わせしない Note over AgentSearch,PolicyGuard: コンプライアンス/GDPR 区分のみ Cloud Run 側で二重チェック Browser->>Gemini: 検索クエリ送信 Gemini->>Gemini: session の IdToken (JWT: Okta claims含む) を検証 Gemini->>STS: IdToken (JWT: Okta claims含む) を workforce identity の認証情報に変換要求 STS-->>Gemini: 結果取得 Gemini->>AgentSearch: Workforce Identity の認証情報で検索要求 AgentSearch->>DataStore: ACL を適用して対象ドキュメントを検索 DataStore-->>AgentSearch: ACL フィルタ済み結果 AgentSearch-->>Gemini: 検索結果 alt 高機密区分（コンプライアンス/GDPR）を含む Gemini->>PolicyGuard: 検索結果の二重チェック要求 PolicyGuard-->>Gemini: 二重チェック済み結果 end Gemini->>Gemini: 検索結果をもとに回答生成 Gemini-->>Browser: 回答と検索結果ページ返却
